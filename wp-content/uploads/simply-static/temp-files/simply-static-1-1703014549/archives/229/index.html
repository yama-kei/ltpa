2010年のレース結果

2010年9月12日 by yama-kei

ちょっぴり気が早いけど、2010年のレース結果のまとめ。

2010年はチームCBに加入しての初年度だったので、ロードレースの経験を積みつつヒルクライムで結果を出そう、というのがひとつの方針。狙うレースと狙わないレースをはっきりさせながら、出られる限りレースに出ようと考えた。

トレーニング本をいろいろ読んだ結果、”Base Building for Cyclist“が一番しっくりきたので、これを参考にしながら6月頭のMt.富士HCに一つ目のピークを持ってこられるようにトレーニングメニュー＆スケジュールを組んで、12月頭からトレーニングを開始・・・。

トレーニングレースとして参加した埼玉クリテやもてぎの100kmマラソン、それにウィンターロードでは、期待していた通りにロードレースのタフさとテクニック・戦略の重要性を学ぶことができた。

埼玉クリテの第二戦では自分のミスで落車を起こしてしまい、ロードレースでは「安全第一」であることを心に刻んだ。JCRCの第一戦＠川越ではこの経験をうまく生かせなかったものの、JCRCの第二戦＠修善寺では得意なコースということもあって自転車レースで初めての勝利。

ヒルクライムでは、6月のMt.富士HCをひとつの目標にしていたので、4月の八ヶ岳や表富士は「どのくらいやれるかな」という気分で参加。6月のMt.富士HCでは目標(1:05)に近い好タイムを出して入賞することができた。今シーズン全体を通して、この時が一番調子がよかったと思う。

その後は夏に向けて一旦コンディションを落とす予定だったにも関わらず、レースが続いていたのでついつい強度の高いトレーニングを重ねてしまって大失敗・・・。結果、オーバートレーニングになった挙げ句に膝を痛めてしまい、ひたすら下降線を辿るシーズン後半になってしまった。

調子の悪かった三宅島、ダメダメだった美ヶ原・・・とか言いつつ一応入賞はしているのだけど、よい状態の走りのイメージが頭にあるので、思った通りに走れないのは大いに悔しい。満足の走りができなくて、入賞できてもあんまり嬉しくない・・・なんて心境になれるとは、去年の自分からは想像もできなかった変化。

7月に入った時点でコンディションがボロボロで、膝痛もすぐには治せないと分かったので、9月頭の秩父宮杯をラストレースにすることを決定。

残りのレースは、コースを覚えるために参加したJCRC第六戦＠群馬CSCや、旅っぽい雰囲気を楽しむための鳥海山。それに、「マウンテンサイクリング」を楽しんだ乗鞍・・・と、消化レースっぽい雰囲気。それでも、秩父宮杯はラストレース＆好きなコースだったので、絶対に最後まで勝負集団に絡むことを意識して、ヘボい走りながらも入賞に絡むことができた。

＊＊

結局、エントリーしていたのにDNSになってしまったのは埼玉クリテの最終戦(冷たい雨)と、膝を痛めた翌週の東京ヒルクライム奥多摩ステージのみ。なんだかんだ言って8月まで月2本ペースでレースを消化してきたことになる。

いろいろとレースに出てみて、好きなレースや得意なコースの傾向が分かってきたかなーという感じ。自分はまだクライマーっぽい走りしかできていないので、本格的にロードレースで戦うにはもっと強度の高い練習を取り入れていかないとだめなことも分かってきた。

(レースじゃないけど、今年もAttack! 299は楽しかった！やっぱり自分は仲間とじゃれ合いながらサイクリングするのが好きみたい)

何はともあれ、今年は膝のリハビリが終わるまでレースはお休み。9月、10月は他の活動でいっぱいいっぱいなので、レースに復帰できるとしても11月になりそうだ。
