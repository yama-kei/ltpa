ウィンターロード第一戦 (観戦)

2011年1月31日 by yama-kei

日曜日は修善寺で行われるウィンターロードへ。

昨年はCクラスで「自転車競技で初めての入賞」ができた自分にとって相性の良いレース。今年はトレーニングのためのレースということでBクラスにエントリーしていたのだけど、長らく風邪をひいてしまった後でボロボロのコンディションだったのでレースはスキップ。BearBellメンバーの仲間が3人参加するということで、応援＆撮影＆レース後のサイクリングするべくKOHさんの車に乗せてもらって修善寺へ。

8:30頃に到着したのでまったり自転車を組む。気温はそこまで低くないけど、風があって寒い。Edge 500が装着されてないことに気づいてテンションダウン。少し遅れてダジャさん、あまがえるさんも到着。この二人がCクラスで、KOHさんがBクラスで走る予定。KOHさんはロードレース初経験だけど、ソリッドな登坂力と上手な下りテクニックをもっているのでけっこうイケるんじゃないかなーと期待。

入場して受付して試走。せっかくなので、参加賞(靴下)とパンフレットだけもらってDNS扱いにしてもらっておく(笑)。KOHさんとあまがえるさんと一緒に試走を2周。自分なりの注意事項やTipsを教えながら1周11分くらいのまったりペースで走って試走終了。うーん、やっぱり修善寺は道が広いので走りやすくてよいですね。

＊＊

スタート地点に戻り、カメラマン装備を整えて撮影。Aクラスはヨーロッパに武者修行に行くという若者が1周目から他を圧倒する強さを見せつけて独走。Bクラスは6,7人強いメンバーがいて、それに追いすがる12,3人のメンバーがしぶとくついていって・・・という展開。Cクラスははじめからバラけ気味で、周を重ねるごとに自然とリストラが進んで・・・という展開。走力・技術がバラバラだと集団が安定しないので、先頭以外のメンバーはみな個人TTに近い状態になっていたように見えた。

・・・ってわけで、撮影した写真はここに置いておきます。常識的な範囲で、自由に使ってもらって構いません。

1-3周目

4周目以降

＊＊

Cクラスの二人は2周目に入るところから先頭から離れてしまって早い段階からレース終了・・・。BクラスのKOHさんは常に強い5,6人のメンバーに混じる形で周回を重ねていく。チームYOU CANがメンバーを揃えていて、そのうち突出した力を持ってる１人が終始レースをコントロールしているように見えた。

KOHさんは(自分のアドバイス通りに)控えめな走りで最終周まで積極的には動かず。最後は10人前後の勝負から完全に抜け出したYOU CANの若者がかっこよく独走勝利。KOHさんは2位争いの3人スプリントに絡めなかったものの、そのすぐあとの5着。初めてのロードレースでここまで走れるのは凄いです。おめでとうございます！

＊＊

Bクラスの表彰式を見てから駐車場に戻る。家族サービスのあまがえるさんとダジャさんとはここでお分かれし、KOHさんと二人で伊豆サイクリングへ出発。

ひとまず修善寺駅方面に下り、駅前の商店街にあるおそば屋さん「紅ほうずき」でランチ。アテがないまま探したお店だったのだけど、ここは当たり。磯辺蕎麦をいただいてお腹を満たしてから南へ向けてサイクリング開始。太い道でも交差点・交通量が少ないので走りやすい。

途中からアップダウンのある狭い道に入たりして南下を続けていく。逆風がきついけど、レースを走ってない自分が先頭固定でマイルドなペースでひく。途中からスパルタンな峠道っぽいところに入りつつ、谷間にわさび田が広がる素敵な景色を楽しんだりしながら淡々とサイクリング(Edge 500がなかったのでコースがわからない(笑))。

ミニ峠を越えてから一気に下って国道に合流し、下り基調の道を追い風に助けられる形で爽快にかっとばして修善寺まで戻り、CSCへの緩やかな登りをクリアーして駐車場に帰着。距離・時間・強度ともに、ちょうどよいサイクリングを楽しむことができて満足できた。

帰りは渋滞に悩まされながらも6時過ぎに東京に着いたので、近所のもつ鍋屋で夕食を食べて解散。全く走れないわけではなかったので、レースに出てもよかったかなーと思ったけど、「頑張れる気がしない時は無理しない」で正解だったかな。

次の第二戦は出られるようにコンディションをあげていくことにしよっと。

カテゴリー: ロードレース パーマリンク
